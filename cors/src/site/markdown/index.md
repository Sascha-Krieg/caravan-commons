## About Commons CORS Filter

Servlet filter for managing CORS header in HTTP responses.


### Maven Dependency

```xml
<dependency>
  <groupId>io.wcm.caravan</groupId>
  <artifactId>io.wcm.caravan.commons.cors</artifactId>
  <version>0.5.0</version>
</dependency>
```
