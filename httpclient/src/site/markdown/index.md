## About Commons HTTP Client

Provides preconfigured Apache HTTP Client instances.


### Maven Dependency

```xml
<dependency>
  <groupId>io.wcm.caravan</groupId>
  <artifactId>io.wcm.caravan.commons.httpclient</artifactId>
  <version>0.2.0</version>
</dependency>
```
